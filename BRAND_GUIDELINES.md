# Magic Land Brand Guidelines

## Logo System

Magic Land uses a minimal, colorful amusement-park identity with three aligned assets:

- `public/logo-mark.svg` - compact circular mark for favicons, app icons, social profile images, stickers, and small badges.
- `public/logo-lockup.svg` - primary horizontal logo for website headers, footers, tickets, posters, and digital layouts.
- `public/favicon.svg` - browser favicon using the same circular mark artwork.

The circular mark intentionally does not include the words "Magic Land." It contains only the park world: ferris wheel, coaster, hot air balloon, carousel, clouds, and sparkles. The name appears in the wordmark beside it, with "Family Fun Park" as the descriptor.

The client-original direction is preserved separately:

- `brand-assets/magicland logo.jpeg` - client-provided original Magic Land Fun Park logo artwork.
- `public/magicland-logo-original.jpeg` - website-ready copy of the same original artwork.
- `public/magicland-logo-transparent.png` - transparent extracted mark used in the website header beside the typed `Magic Land / Family Fun Park` lockup.
- `brand-assets/magicland-logo-transparent.png` - transparent PNG handoff file for design/social use.

Do not delete the newer `logo-mark.svg` / `logo-lockup.svg` system; keep it available as the proposed modern identity while the client-provided original logo is used on the website.

## Logo Usage

- Use the horizontal lockup whenever space allows.
- Use the circle mark only when the wordmark would be too small to read.
- Keep clear space around the logo equal to at least half the circle mark diameter.
- Do not stretch, rotate, recolor, add extra shadows, or place the logo on busy images without a white or very light backing.
- Minimum digital size:
  - Lockup: 160px wide
  - Mark: 32px wide

## Color Palette

Primary colors:

- Magic Navy: `#030D46`
- Park Blue: `#1F8BEA`
- Joy Red: `#EF2B5A`
- Ride Orange: `#FF8A00`
- Play Green: `#33B84A`
- Sunshine Yellow: `#FCD447`

Supporting interface colors:

- Background: `#FBF8FF`
- Soft Surface: `#ECECFF`
- Text: `#0F1842`
- Muted Text: `#46464F`
- Border: `#C6C5D1`

## Typography

Current website typography:

- Headlines and brand-adjacent display: `Quicksand`
- Body, labels, forms, and navigation: `Plus Jakarta Sans`

Brand voice should feel joyful, clear, family-safe, and welcoming. Avoid overly technical phrases on guest-facing pages.

## Visual Style

- Keep the logo bright and playful, but avoid clutter in small UI areas.
- Use real attraction imagery for content cards.
- Use colorful accents sparingly so the site remains light and easy to scan.
- Prefer rounded but controlled shapes; the UI should feel friendly without becoming childish.

## Messaging

Core line:

> A place where kids laugh, families bond, and memories become magic.

Supporting language should highlight:

- VR games and simulator fun
- Family rides and arcade challenges
- Creative Village activities like pottery, painting, colors, and arts
- Warm hospitality and easy family planning
